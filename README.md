# DocDrift AI

AI-powered documentation generator with doc-drift detection.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
Install the project dependencies (including the Vercel CLI helper):
```bash
npm install
```

### Running Locally
To run the full stack locally (including Vercel Serverless API Functions and the static frontend):
```bash
npm run dev
```
This will start the Vercel development server, typically available at `http://localhost:3000`.

*Note: If you only want to serve the static frontend without running any backend API endpoints, you can use:*
```bash
npx serve public
```
